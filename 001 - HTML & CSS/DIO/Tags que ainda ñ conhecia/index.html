"div"

Muito útil para organizar o código e separar o que tá no 'menu' do que tá no 'background' por exemplo, mas pode ocupar bastante espaço;

"Span"

Muito parecido com o div, porém ocupa menos espaço, bem útil quando se quer separar apenas uma série de palavras, ou só 1 caractere (se é que você precisa fazer isso);

"fieldset"

Mesmo apenas no html, essa tag pode deixar tudo mais atraente e organizado;

"iframe"

Te permite abrir uma janela de outro site na aba presente. Cuidado! Pode afetar a segurança.

"track"

Adiciona legendas em vídeos;

Tabelas

"table"

Reserva espaço para dados tabulares;

"tr"

Significa "table row" ou linha de tabela

"td"

Colunas ou células de uma tabela;

"th"

Adiciona cabeçalho na tabela;

"tbody"

Organiza todo o componente da tabela juntando todas as tags referente a tabela no código;

"thead"

Organiza todo o componente de cabeçalhos dentro da tabela, pode ficar dentro do tbody também;

"tfoot"

Organiza todo o componente de rodapés dentro da tabela, pode ficar dentro do tbody também;

"caption"

Descrição ou tema usado para tabelas;
