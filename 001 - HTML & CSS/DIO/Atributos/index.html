Atributos comuns e não comuns, mas úteis para tags!

Para outras linguagens

  • "Style": Vale somente pra css, utilizado de diversas maneiras, tipo mudar cor de uma tag em especifico;
  • "ID": Usado para obter atribuições em Java Script.

Atributos mais comuns

  • "Type": Altera o tipo da tag ou elemento, nele podemos colocar o tipo:'number', 'color', 'text', 'password', 'submit', etc;
  • "Src": Procura o que a tag busca, como no caso da tag img, ele vai buscar um arquivo de imagem;
  • "Width": Altera Largura;
  • "Height": Altera Altura;
  • "Name": Nomeia outros atributos, elementos, tags, etc;
  • "Action": Direciona os dados para o endereço colocado;
  • "Method": Ele te 2 valores possíveis, o 'GET' que salva os dados adicionados, mas sem seguraça, já o 'POST' os salva com segurança;
  • "Target": Atribui ao clica no link a maneira como a nova aba vai aparecer, ao colcoar o valor '_blank' junto de target ele vai abrir numa nova aba;
  • "Autocomplete": Armazena dados já indicados e quando repetidos completa-os se o valor atribuído for 'on'.
  • "On...": São eventos, muito usados em JS;
  • "border": Adiciona bordas.
  • "align": Alinha o conteúdo de acordo com o elemento (center, etc);

Elementos

  • "radio": Botão de seleção, que ao receber o mesmo nome só te deixa escolher 1 opção;
  • "Checkbox": Adiciona caixa de checagem de forma opcional, colocando [] ele se torna uma lista;
  • "Input": Determina ação, geralmente usado para criar formulários;
  • "Button": Cria um botão... Tipos de botão:
    1. button type="button": Cria um botão clicável, mas sem ação. A ação pode ser aplicada em JS depois;
    2. button type="reset": Limpa os campos que estão dentro de um formulário;
    3. button type="submit": Envia os dado;
  • "Select": Adiciona uma caixa de seleção... Tipos de select:
    1. select type="role": Coloca opções na seleção quando junto da tag option;
  • "Textarea":Adiciona uma área para escrever redimencionavel;
  • "summary": Adiciona um sumário somente vísivel no código, geralmente usado pra explicar algo;
