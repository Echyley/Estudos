Trabalhando com Listas

Listas Ordenadas

Usamos a tag "Ol" para criar uma ordenação nos items, e entao usamos a tag "Li" para criar uma lista:

  1. Acordar;
  2. Tomar café;
  3. Trabalhar;
  4. almoçar.

Tambem podemos usar dentro da tag "Ol" a semântica "Type" para indicar em qual numero a lista começara, ou se será uma lista com letras alfabéticas, dá até com numeros romanos!
Veja:

  1. ...
  2. ...
  3. ...
  1. ...
  2. ...
  3. ...
  1. ...
  2. ...
  3. ...

Listas não Ordenadas

Aqui a sequencia não precisa fazer sentido, geralmente usamos quando precisamos comprar algo no mercado. Usamos a tag "Ul"
Também podemos usar a semântica "Type" debtro da tag Ul para mudar o simbolo dela, com "circle", "Square" e "disc".

  • Leite;
  • Coxinha;
  • Tomate;
  • Arroz.

Misturando listas

Pode-se misturar as listas e suas formas da maneira que quiser.
Veja:

Ol com Ul

Animes favoritos

  1. Antigos
    • Samurai X;
    • Astro Boy;
    • One Piece.
  2. Novos
    • 86;
    • World Trigger;
    • Dr.Stone.

Jogos Favoritos

  1. Snes
    • Donkey Kong;
      • DK Country;
      • Dk 2;
      • DK 3.
    • Super Mario;
    • Lion King.

  2. Playstation 2
    • Gta San Andreas;
    • Rattouile;
    • Lego Star Wars.

  3. Pc
    • Minecraft;
    • Roblox;
    • The Sims.

Para fazer isso, basta colocar uma tag "Ol" (ou "Ul" se preferir) após uma "Li" dentro da primeira "Ol"/"Ul"

Lista de Definição

Usamos a tag "Dl" para criar uma lista de definição, "Dt para criar um termo e "Dd" para criar uma definição.
