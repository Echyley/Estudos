Usando links externos (Que não pertencem a você)

Para ancorar alguma frase ou palavra e direciona-la a um lugar especifico basta colocar a tag "a" e colocar na type "href" sua URL.

Para manter sua guia atual aberta ao clicar no link basta colocar o elemento target="_blank" rel="external" dentro da type já existente "Href".
E para abrir o conteúdo da próxima página na guia atual basta colocar o elemento Target="_self" dentro da type já existente "Href".

Usando links internos (Que nos pertence, ou seja, está em nosso servidor)

Falando na semântica, para indicar qual é a próxima página e qual é a anterior coloca-se o elemento rel="next" dentro da type já existente "Href", usado para indicar a página seguinte e usamos o elemento rel="prev", para indicar a página anterior. Isso é importante, pois desse modo o navegador não vai se confundir.
